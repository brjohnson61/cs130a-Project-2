To compile main, type "make" at the terminal

>make

To run the compiled main, type ./MainProgram

>./MainProgram

After this:
    Entering 1 will prompt a word to search
    Entering 2 will prompt a word to insert
    Entering 3 will prompt a word to delete
    Entering 4 will sort the current words and place them in output.txt with one line between hash table and binary search tree outputs
    Entering 5 will prompt two words and will print out all words in between those two words.

Press Ctrl+C to quit.
